# Privacy Policy

**GEDCOM DNA Match Finder**

*Last updated: May 2026*

---

## Overview

GEDCOM DNA Match Finder is a desktop application that processes your GEDCOM data entirely on your local computer. It does not transmit GEDCOM data to any server and does not collect any personal information. It does not modify your GEDCOM file.

---

## Data You Provide

The application reads GEDCOM files that you select from your local filesystem. These files may contain personal information such as names, dates of birth, family relationships, and DNA match flags.

All processing of this data occurs locally on your device. No data from your GEDCOM files is ever sent to any external service, server, or third party.

---

## Local Storage

The application stores the following information on your device only:

- **Settings** — your preferences (theme, font size, search defaults) saved in a local configuration file.
- **Recent files** — paths to recently opened GEDCOM files, stored in the same configuration file for convenience.
- **Cache** — a parsed representation of your GEDCOM files stored locally to speed up subsequent loads. This cache may contain names, dates, relationship data, DNA marker details, and raw GEDCOM record content such as notes, places, source references, contact details, or other fields present in your GEDCOM files. You can delete it at any time via **Menu → Preferences → Clear Cache** or **Menu → Clear cache…**

All local storage is confined to your device and is never shared externally.

---

## Network Access

This application has no analytics, no telemetry, no crash reporting, and no automatic update checks. With the one exception below, it never accesses the network.

If you choose **Help → Check for updates…**, the application makes a network request to GitHub to ask for the latest public release of GEDCOM DNA Match Finder. This request does not include your GEDCOM file, your settings, your recent files list, or your cache contents. GitHub receives the normal technical information involved in any web request, such as your IP address and user agent.

---

## Third-Party Data

If your GEDCOM file contains data about living individuals or data sourced from third-party services (such as AncestryDNA), please ensure you have appropriate rights and permissions to use that data in accordance with those services' terms and your applicable privacy laws.

---

## Children's Privacy

This application does not knowingly collect or process data from children. All data processed by the application comes solely from GEDCOM files you provide.

---

## Changes to This Policy

This privacy policy may be updated with new releases of the application. The current version is always included with the application.
